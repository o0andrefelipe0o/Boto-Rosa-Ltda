# Projeto Sistema de Gestão de Transportes – Boto-Rosa Ltda

[![SENAI](https://i.imgur.com/jxmOeKW.png)](https://www.fiemg.com.br/senai/)

Este repositório contém um projeto acadêmico desenvolvido no **SENAI CTTI – Belo Horizonte/MG**, como parte do curso de **Lógica de Programação em C**.
O sistema foi projetado para atender às necessidades de gestão de transportes fluviais da empresa fictícia **Boto-Rosa Ltda**, responsável pelo transporte de passageiros e cargas pelo Rio Amazonas até portos marítimos da costa brasileira.

---

## Sobre o Projeto

Este foi **meu primeiro projeto de desenvolvimento de sistemas**, elaborado ainda no início dos meus estudos.
Na época do desenvolvimento, os conhecimentos estavam restritos com **estruturas condicionais (`if`, `else if` e `else`)**, não havendo ainda domínio de conceitos como **laços de repetição (`for`, `while`)**, vetores ou estruturas mais avançadas.

Mesmo com essas limitações, o projeto cumpre seu papel acadêmico ao aplicar, na prática, os conceitos aprendidos até aquele momento.

---

## Objetivos

* Aplicar os fundamentos iniciais de lógica de programação em C
* Desenvolver um sistema funcional com menus estruturados
* Implementar cadastros, listagens e exclusões básicas
* Simular um sistema real de gestão de transportes

## Como Executar o Projeto

### Pré-requisitos

* Sistema operacional Windows

### Execução

1. Clonar o repositório:

```bash
git clone https://github.com/o0andrefelipe0o/Boto-Rosa-Ltda
```

2. Abrir o executável:

* Pode ocorrer de o sistema exibir uma mensagem solicitando confirmação para executar o programa, informando que não é possível garantir sua confiabilidade.

3. Executar o programa:

* O sistema inicia na tela de login.
